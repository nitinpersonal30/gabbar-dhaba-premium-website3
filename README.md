# Gabbar Dhaba & Family Restaurant — Premium Multipage Website

A professional, responsive, multi-page restaurant website concept for **Gabbar Dhaba & Family Restaurant, Sultanpur Road, Lucknow**.

## Pages Included

- `index.html` — Premium home page with hero slider, signature dish and booking CTA
- `about.html` — Restaurant story, food policy and values
- `menu.html` — Complete menu with search, category filters and WhatsApp item enquiry
- `services.html` — Restaurant, A/C banquet hall, lawn and A/C room booking sections
- `gallery.html` — Category-based photo gallery with lightbox
- `contact.html` — Contact cards, WhatsApp booking form and Google Maps embed

## Features

- Fully responsive desktop, tablet and mobile design
- Sticky header and mobile menu
- Hero image slider using real restaurant photos
- Full menu categories and all available items from the official menu page
- Menu search and filters
- Add menu items to WhatsApp enquiry list
- Gallery filters: Exterior, Dining, Lawn, Banquet, Food
- Gallery lightbox image preview
- WhatsApp booking forms for table, food, banquet, lawn and A/C rooms
- Call, WhatsApp and Google Maps buttons
- SEO meta tags and Restaurant schema markup
- Mobile bottom CTA bar

## How to Use

1. Extract the ZIP.
2. Open `index.html` in your browser.
3. Upload all files to GitHub repository.
4. Go to GitHub repository **Settings → Pages**.
5. Select branch `main` and folder `/root`.
6. Wait 2–5 minutes for GitHub Pages live link.
